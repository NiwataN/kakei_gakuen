## エレベーターピッチ
残高把握と既存のアプリの問題点を解決したい
お金の管理が苦手な人向けのめんどくさくない家計簿を実現する家計簿アプリです
これはかんたんで多様な入力方法ができ、既存の家計簿アプリとは違ってとにかく入力が楽で、残高をゲーム的に管理することができます
